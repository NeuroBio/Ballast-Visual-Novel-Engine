Misc. Notes

	
    
		
  • "Customizable Actions" points out when the engine is calling a callback provided to it at construction.
    • 
		
  • You may notice that Restart Scene does not call the callbacks that Start Scene does; restart scene is working off of cached data.
    • 
		
  • The engine keeps track of cumulative save data changes itself.  It provides the save data changes each beat incurs in case there are display needs for the data (e.g. show a banner for unlocking an achievement).
    • 
		
  • The only scene data the engine keeps track of itself is which characters were added and removed.
